Doozy

Web Application

View on GitHub

Doozy is a streamlined to-do list application designed to provide hands-on experience with the Quasar Framework and Axios library, while demonstrating core front-end development concepts. Built using the Quasar CLI, the app integrates seamlessly with the JSONPlaceholder API to perform CRUD operations. Users can add new tasks via a simple form featuring an input field and submit button, and manage their tasks through a responsive and interactive interface. Axios handles all HTTP requests to the API, including retrieving existing tasks, submitting new ones, editing task details, and deleting entries.

The primary goal of Doozy is to illustrate practical usage of API integration and dynamic UI updates within a modern Vue.js-based framework. As tasks are created, updated, or deleted, the application reflects changes in real time, enhancing the user experience.
