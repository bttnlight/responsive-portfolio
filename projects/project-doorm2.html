Doorm

Website

Doorm is a real estate-style mobile application built as a team project for our File Processing and Database Systems class. Tailored for students in Cebu, the app allows users to either list dormitories or apply as tenants, streamlining the search and management of student accommodations. The platform provides features such as dorm listings, application tracking, and profile management, focusing on user convenience and data-driven functionality.

On the frontend, I built the user interface using React and styled it with Tailwind CSS. I translated our design prototypes into interactive, responsive components, handling core functionalities such as listing displays, form submissions, filtering, and navigation. I ensured consistent styling, clean component structure, and smooth user interactions to deliver a polished and responsive web experience.

Although backend development was primarily handled by other teammates, I contributed to some backend tasks using Django, particularly involving basic CRUD operations and database model integration. I also assisted in ensuring proper data flow between the frontend and backend through API integration and testing.
