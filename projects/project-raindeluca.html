Rain De Luca: In Action!

Game Development

View on GitHub

Rain de Luca: In Action is a narrative-driven courtroom game inspired by Ace Attorney, designed around a Deterministic Finite Automaton (DFA) framework. Each state in the DFA maps to a key plot or decision point, guiding players through branching storylines based on their choices. The game features dynamic scene management, a robust dialogue system, evidence tracking, and immersive courtroom mechanics to create an engaging, interactive legal drama.
