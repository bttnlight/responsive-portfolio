Ballpoint

Mobile App

Ballpoint is an AI-enhanced note-taking application developed as a team project for our Software Engineering class. The app aims to streamline the learning process by automating note extraction from images and audio, organizing content intelligently, and offering tools for querying, autocomplete, advanced searching, and summarization. It is designed to improve the efficiency of studying, allowing users to focus more on understanding and less on manual note-taking.

As one of the frontend developers, I worked on implementing the UI using React Native. I translated our design prototypes into responsive, functional components and handled user interactions for key features such as note creation and category management. I also managed state and API integration with our Django backend, ensuring smooth data flow and responsiveness. My focus was on maintaining a consistent user experience, minimizing latency, and handling errors gracefully to support a reliable and polished frontend.
