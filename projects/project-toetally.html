Toetally

Web Application

View on GitHub

Toetally is a personal finance tracking web application built with Django, designed to help users manage their income and expenses with ease and clarity. The application provides secure user authentication, enabling individuals to register, log in, and maintain their own private financial data. Utilizing Djangoâ€™s form handling and model capabilities, users can create, read, update, and delete income and expense records. Each financial entry includes a title, amount, date, type (income or expense), and optional notes, with expenses further categorized (e.g., Food, Travel, Bills) for detailed tracking.

The core of Toetally is its intuitive dashboard, which offers users a clear monthly financial overview. This includes a real-time summary of total income, total expenses, and remaining balance, accompanied by interactive visualizations powered by Chart.js. A pie chart displays expense distribution by category, while a bar graph tracks income versus expenses over time. These insights empower users to make informed financial decisions. With a seamless UI for data entry and an emphasis on usability, Toetally combines backend robustness with engaging frontend visuals to deliver a complete personal budgeting experience.
